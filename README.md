# 배움 놀이터 🌟 (한글 받아쓰기 & 구구단 15단)

어린이들을 위한 한글 맞춤법 학습과 직관적이고 세련된 곱셈 연산 훈련을 동시에 제공하는 통합 학습 플랫폼입니다.

---

## 🎯 핵심 앱 기능

### 1. 한글 받아쓰기 챌린지 📝
- **학년 선택**: 1학년 / 2학년 선택 가능
- **급수별 문제**: 학년별 8~10단원, 단원당 10문제의 고밀도 구성
- **TTS 음성 재생**: Web Speech API 기반 정밀 발음 구현 (자동 재생 + 다시 듣기 버튼)
- **정오답 시각적 강조**: 실시간 피드백 및 결과 분석 화면 제공

### 2. 구구단 15단 챌린지 🧮
- **단수 커스텀 설정**: 2단부터 15단까지 다중 선택 토글 지원 및 퀵 프리셋 제공
- **하이브리드 모드**: 4지선다형 **객관식** 및 모바일 친화형 가상 키패드가 탑재된 **주관식** 선택
- **수학적 오개념 오답 보기 알고리즘**: 인접 단수 곱셈 에러, 사칙 연산 혼동, 자릿수 전치 등 학습자의 인지 오류 경향을 반영한 정밀 보기 자동 출제
- **실시간 오답 노트**: 결과 화면에서 문제당 걸린 속도 통계와 오답 노트를 바로 복습할 수 있는 기능 제공

---

## 🛠 기술 스택

- **Core**: React 19 + Vite 7
- **Styling**: Vanilla CSS (글래스모피즘, 프리미엄 다크/수학 테마 및 네온 Glow 효과)
- **Web API**: Web Speech API (TTS)

---

## 📂 디렉토리 구조

```
src/
├── App.jsx                       # 통합 라우터 (앱 선택기)
├── App.css                       # 공통 스타일 및 런처 스타일
├── main.jsx                      # 엔트리포인트
├── index.css                     # 글로벌 CSS (리셋, 폰트)
├── utils/
│   └── gugudanUtils.js           # [NEW] 오답 생성 및 문제 셔플 알고리즘
├── components/
│   ├── GradeSelector.jsx         # 학년 선택 화면
│   ├── LevelSelector.jsx         # 급수(단원) 선택 화면
│   ├── Game.jsx                  # 받아쓰기 게임 진행 화면
│   ├── Question.jsx              # 받아쓰기 개별 문제
│   ├── Results.jsx               # 받아쓰기 결과 화면
│   └── gugudan/                  # [NEW] 구구단 15단 서브 모듈
│       ├── GugudanSetup.jsx      # 단수 및 풀이 모드 설정 화면
│       ├── GugudanGame.jsx       # 구구단 진행 엔진 및 가상 키패드
│       ├── GugudanResults.jsx    # 결과 보고서 및 오답 분석 뷰
│       └── Gugudan.css           # 구구단 전용 글래스모피즘 스타일시트
└── data/
    ├── gradeData.js              # 학년별 받아쓰기 데이터 인덱스
    ├── dictationData.js          # 1학년 받아쓰기 데이터
    └── dictationData2.js         # 2학년 받아쓰기 데이터
```
